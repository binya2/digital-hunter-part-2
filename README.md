️Digital Hunter API
מערכת לניתוח נתוני מודיעין בזמן אמת, זיהוי דפוסי תנועה וויזואליזציה על מפה.

 איך מריצים?
ודא שאתה בתיקיית השורש של הפרויקט.



docker compose up --build

המערכת תהיה זמינה בכתובת: http://localhost:8000.

תיעוד Swagger: http://localhost:8000/docs.

 מבנה וטכנולוגיות
FastAPI: שרת Web מהיר ואסינכרוני המנהל את הראוטרים של המערכת.

MySQL: מסד נתונים רלציוני לשמירת אותות ומטרות.

Matplotlib & Geopandas: מנוע לייצור מפות PNG דינמיות.
